# Non-Goals

MASH is **not**:

- A traditional filesystem.
- A block‑level storage hypervisor.
- A backup application.
- A sync application.
- A cloud storage provider.
- A replacement for Git.
- A replacement for Everything.
- A replacement for Syncthing.

Instead, MASH **orchestrates** existing tools (SMB, WebDAV, Git, Syncthing, Everything, cloud object stores, etc.) to provide a unified, intent‑driven storage experience.
