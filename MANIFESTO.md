# Manifesto

## Principles and Laws

- **First Law**: MASH must never make user data less accessible than it was before MASH existed.
- **Second Law**: The user expresses intent; the Librarian determines implementation.

### Examples of the Second Law

- User: *Keep three copies.* ? Librarian decides *where* those copies live.
- User: *Archive this.* ? Librarian selects an appropriate archive tier.
- User: *Keep this project hot.* ? Librarian chooses a fast storage provider.

The system is a **Storage Hypervisor with a Digital Librarian control plane** – the hypervisor provides the infrastructure, the Librarian provides the intelligence.
